# Chroma.js::Rails

A Gem for Chroma.js library<br>
This Gem is made to support the use of Chroma.js library with Rails Assets

## Installation

Add this line to your application's Gemfile:

    gem 'chroma.js-rails'

And then execute:

    $ bundle

Or install it yourself as:

    $ gem install chroma.js-rails

## Usage

### CoffeScript
    #= require chroma

### JavaScript
    //= require chroma
